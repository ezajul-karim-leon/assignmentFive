Question-1: What are the different ways to select an element in the DOM?

There are several type of way to select an element in the DOM. Such as, 1. Id Selector, 2. class Name Selector, 3. Query Selector, 4. Tag Name Selector

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference is : 1. innerHTML sets the HTML content of an element, 2. InnerText sets the text content of an element and Ignores hidden elements and collapses whitespace. 3. TextContent sets the text content of an element Includes hidden elements' text and preserves whitespace.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript that allows you to handle events more efficiently by using a single event handler for multiple elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the DOM (Document Object Model) where an event that is triggered on a child element propagates or bubbles up through its parent elements.
